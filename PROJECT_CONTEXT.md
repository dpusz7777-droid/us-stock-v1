## 二、项目目标

本项目计划开发一套适合美股投资新手使用的 AI 研究和监控系统。

系统主要目标：

1. 管理自选股
2. 获取美股行情
3. 筛选值得关注的股票
4. 监控相关新闻
5. 分析新闻的利好、利空和重要性
6. 解读公司财报
7. 生成每日市场晨报
8. 记录个人持仓和交易
9. 提供风险提示
10. 未来支持 Telegram 或其他渠道推送
11. 未来预留券商 API 接口
12. 暂时不自动执行真实交易

本项目的定位是：

* 投资研究辅助工具
* 信息整理工具
* 风险提示工具
* 投资学习工具

本项目不承诺收益，也不保证预测准确。

## 八、技术原则

开发时应遵守：

1. 优先使用 Python。
2. 优先使用稳定、常见的开源库。
3. 不引入不必要的复杂框架。
4. 每个模块应职责明确。
5. 配置与代码分开。
6. API Key 使用 `.env`。
7. `.env` 不得上传 GitHub。
8. 所有外部数据必须记录来源。
9. 所有网络请求必须有超时和错误处理。
10. 所有重要程序必须输出清晰日志。
11. AI 分析结果不得冒充确定事实。
12. 投资结论必须包含风险提示。
13. 暂时不开发自动真实下单。
14. 所有真实交易必须人工确认。
