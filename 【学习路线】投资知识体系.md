# 美股投资新手学习路线

## 阶段一：基础知识（第1-2周）

### 1.1 股票市场基础
- **纳斯达克市场特点**
  - 科技成长股集中地
  - 波动性较大（适合学习）
  - 流动性充足
  - 交易时间：美国东部时间 9:30-16:00

- **必知概念**
  - 股票 = 公司的一部分所有权
  - 股价 = 市场对公司价值的估值
  - 买卖差价 = 买价和卖价的差异
  - 市场情绪 = 供求关系驱动

### 1.2 投资风格类型
| 风格 | 特点 | 适合新手? |
|------|------|---------|
| 价值投资 | 买便宜的好公司，长期持有 | ✅ |
| 成长投资 | 买高成长公司，接受高估值 | ✅ |
| 动量投资 | 跟风上涨趋势 | ❌ 容易被套 |
| 短线交易 | 频繁交易赚差价 | ❌ 手续费过高 |
| 指数投资 | 买整个市场指数 | ✅ 简单稳健 |

**建议**：前6个月采用 **价值+成长** 的混合风格

## 阶段二：财务分析（第2-4周）

### 2.1 必须懂的三个财务报表

#### 1️⃣ 资产负债表（Balance Sheet）
**回答问题**："公司拥有什么？欠债多少？"

| 指标 | 含义 | 好的标准 |
|------|------|---------|
| 总资产 | 公司拥有的所有资产 | 增长趋势 |
| 总负债 | 公司欠的债 | 负债率<60% |
| 股东权益 | 公司净资产 | 稳定增长 |
| 流动比率 | 短期偿债能力 | >1.5 |

#### 2️⃣ 利润表（Income Statement）
**回答问题**："公司赚了多少钱？"

| 指标 | 含义 | 好的标准 |
|------|------|---------|
| 营收(Revenue) | 总收入 | 稳定增长 |
| 毛利率 | (收入-成本)/收入 | 越高越好 |
| 营业利润 | 主营业务赚的钱 | 稳定>0 |
| 净利润 | 税后纯利润 | 增长趋势 |

#### 3️⃣ 现金流表（Cash Flow）
**回答问题**："实际有多少现金进出？"

| 指标 | 含义 | 好的标准 |
|------|------|---------|
| 经营现金流 | 业务产生的现金 | 正数且增长 |
| 投资现金流 | 研发/扩张投入 | 适度负数 |
| 融资现金流 | 借贷/融资活动 | 要看战略 |

### 2.2 关键财务指标速查

```
🔴 估值指标（判断股价贵不贵）
├─ P/E比率（市盈率）= 股价 ÷ 每股收益
│  └─ 解读：<20低估，20-30合理，>40高估
├─ P/S比率 = 市值 ÷ 年收入
│  └─ 解读：关注增长快的公司
└─ PEG比率 = P/E ÷ 增长率
   └─ 解读：<1最好，综合考虑增长

🟢 盈利能力（公司赚钱能力）
├─ ROE = 净利润 ÷ 股东权益
│  └─ 解读：>15%优秀，>10%良好
├─ ROA = 净利润 ÷ 总资产
│  └─ 解读：>8%优秀
└─ 净利润率 = 净利润 ÷ 收入
   └─ 解读：越高越好，但要对比行业

🔵 增长指标（公司成长潜力）
├─ 收入增速 = (今年收入-去年收入) ÷ 去年收入
│  └─ 解读：>20%快速增长，>10%良好
├─ 利润增速 = (今年利润-去年利润) ÷ 去年利润
│  └─ 解读：需要超过收入增速
└─ EPS增速 = 每股收益增速
   └─ 解读：衡量盈利增长

⚫ 安全性指标（风险评估）
├─ 负债率 = 总负债 ÷ 总资产
│  └─ 解读：<50%安全，50-70%可接受
├─ 流动比率 = 流动资产 ÷ 流动负债
│  └─ 解读：>1.5为佳
└─ 利息覆盖率 = EBIT ÷ 利息支出
   └─ 解读：>5倍安全，<2倍危险
```

### 2.3 分析流程
```
第1步：查看公司概况
       ↓
第2步：对比历史财务数据（3-5年趋势）
       ↓
第3步：对比行业同类公司
       ↓
第4步：评估增长可持续性
       ↓
第5步：计算合理估值
       ↓
第6步：得出投资结论
```

## 阶段三：行业和竞争分析（第3-5周）

### 3.1 分析框架（五力分析）

**1. 行业增长** 
- 市场规模和增速
- 行业生命周期阶段
- 政策和技术驱动

**2. 竞争地位**
- 市场占有率
- 品牌认可度
- 成本优势

**3. 竞争强度**
- 竞争对手数量
- 进入壁垒
- 产品同质化程度

**4. 供应商议价力**
- 原料集中度
- 替代品存在
- 供应链稳定性

**5. 客户议价力**
- 客户集中度
- 转换成本
- 产品必需性

### 3.2 纳斯达克主要行业
| 行业 | 代表公司 | 特点 |
|------|---------|------|
| 软件 | MSFT, ADOBE | 高利润率，可持续增长 |
| 芯片 | NVDA, INTC | 景气度周期，高风险 |

## 阶段四：估值和买卖决策（第4-6周）

### 4.1 估值方法

**方法1：相对估值法（最简单）**
```
合理股价 = 行业平均P/E × 公司EPS
```

**方法2：绝对估值法（最准确）**
```
1. 预测未来5年现金流
2. 用折现率(10-12%)折现
3. 得到内在价值
4. 与现价对比：
   现价<内在价值×0.7 → 买入
   现价>内在价值×1.3 → 卖出
```

### 4.2 买入决策清单
- [ ] 公司基本面良好（营收和利润增长）
- [ ] 财务状况健康（负债率合理）
- [ ] 行业前景看好（5年内持续增长）
- [ ] 股价相对合理（P/E在行业平均水平或以下）
- [ ] 竞争优势明确（护城河存在）
- [ ] 我理解这个公司的商业模式
- [ ] 风险可控（不全仓，有止损）

### 4.3 卖出决策清单
- [ ] 基本面恶化（收入利润下降）
- [ ] 行业趋势反转（看不到增长前景）
- [ ] 股价严重高估（P/E远超行业平均）
- [ ] 技术支持被破坏（技术面确认）
- [ ] 止损价被触及（执行既定计划）
- [ ] 投资理由消失（投资逻辑被推翻）

## 阶段五：风险管理和心态（贯穿始终）

### 5.1 资金配置
```
总资金 5,000美元

建议配置：
├─ 现金储备（20%）= $1,000
│  └─ 用于调整仓位和止损
├─ 主要持仓（60%）= $3,000
│  ├─ 2-3只核心股票
│  └─ 每只不超20%
├─ 学习持仓（15%）= $750
│  ├─ 1-2只创新或高风险公司
│  └─ 小额试错
└─ ETF投资（5%）= $250
   └─ 纳斯达克100或标普500指数
```

### 5.2 风险控制要点
1. **单笔交易止损** = 投资额的5-10%
2. **单只股票最大亏损** = 总资金的3-5%
3. **做空/杠杆/期权** = 前1年禁止
4. **集中度风险** = 单股不超20%

### 5.3 心态调整
- **涨时不贪心** → 执行卖出计划
- **跌时不恐慌** → 检查基本面，不盲目割肉
- **频繁交易冲动** → 问自己"为什么买？"
- **损失中学习** → 每笔亏损都要复盘原因
