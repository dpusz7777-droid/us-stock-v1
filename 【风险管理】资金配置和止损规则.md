# 资金管理和风险控制规则

## 🎯 核心风险管理原则

### 原则1：资金配置（CRITICAL）
```
总资金：$5,000

安全配置模式：
├─ 现金储备（20%）$1,000
│  ├─ 用于调整仓位
│  ├─ 用于止损后补仓
│  └─ 用于捕捉低吸机会
│
├─ 主力持仓（60%）$3,000
│  ├─ 持仓1（20%）$1,000 - 核心稳健股
│  ├─ 持仓2（20%）$1,000 - 核心成长股
│  ├─ 持仓3（10%）$500  - 试错学习
│  ├─ 持仓4（10%）$500  - 试错学习
│  └─ 规则：任何单只股票不超$1,000
│
└─ 指数基金（20%）$1,000
   ├─ 纳斯达克100 ETF (QQQ)
   └─ 可作为对标或补充
```

### 原则2：单笔交易风险控制
```
投资额：$500-1,000/只股票
止损距离：5-10%
风险额：$25-100/笔交易
    ↓
控制规则：止损位 = 买入价 × (1 - 止损%)

示例：
- 买入价：$100
- 止损比例：8%
- 止损价：$92
- 如果触及$92，必须卖出
- 这笔交易最多亏$800或更少
```

### 原则3：组合风险评分
| 配置 | 股票集中度 | 行业集中度 | 风险等级 | 预期收益 |
|------|---------|---------|--------|--------|
| 安全型 | 3-5只 | 分散 | ⭐ | 5-10% |
| 平衡型 | 2-3只 | 集中1-2个 | ⭐⭐ | 10-20% |
| 激进型 | 1-2只 | 高度集中 | ⭐⭐⭐⭐ | 20%+ |

**新手建议**：平衡型，逐步学习

## 🛑 止损规则（必须执行）

### 何时必须止损
1. **技术面止损** - 价格跌破支撑位8-10%
2. **基本面止损** - 公司基本面恶化（收入下滑、亏损扩大）
3. **时间止损** - 6个月内没有起色，从观察转为退出
4. **理由止损** - 投资理由消失（产品失败、高管变化等）

### 止损价格表
```
购买价格    止损比例（新手）   止损比例（进阶）
$50-100     8-10%            5-7%
$100-200    8-10%            5-7%
$200以上    10-12%           7-8%

示例：
买$100的股票，止损8%
止损价 = $100 × (1-0.08) = $92

亏损金额 = ($100-$92) × 100 = $800
（如果你持有100股）
```

### 止损纪律
- ⚠️ **禁止**：不设止损、希望股票反弹、加仓摊低成本
- ✅ **必须**：严格执行，不因情绪改变
- 📝 **记录**：每次止损都要记录原因，作为教训

## 📊 投资组合管理

### 持仓检查清单（每周）
- [ ] 是否所有持仓都有清晰的投资理由？
- [ ] 是否有任何持仓跌破止损价？（如是，立即止损）
- [ ] 是否有过度集中的风险？（超过20%？）
- [ ] 现金占比是否在15-25%？

### 月度调整规则
- **不调整**：如果系统持仓都符合以下条件
  - 基本面未变
  - 估值未严重偏离
  - 尚未达到盈利目标

- **调整**：以下情况发生时
  - 基本面发生重大变化
  - 估值严重高估（P/E超过行业平均50%）
  - 完成盈利目标（例如涨幅达到25-30%）
  - 组合过度集中

### 调整示例
```
初始持仓：
- 持仓1：MSFT $1,000 (10%)  ✓ 保留
- 持仓2：NVDA $500 (-8%)    ⚠️ 跌破止损，止损出场
- 持仓3：TSLA $1,200 (+20%) ✓ 继续持有
- 现金：$1,300

调整后：
- 现金用$500配置新机会或加强弱势持仓
- 保持平衡，不过度调整
```

## ⚡ 特殊情况处理

### 情况1：股票暴跌（单日跌幅>10%）
- 👉 **第一步**：保持冷静，不要惊慌
- 👉 **第二步**：查看公司最新新闻，基本面是否恶化？
- 👉 **第三步**：检查业绩和财报，是否值得继续持有？
- 👉 **第四步**：如果基本面未变，这可能是低吸机会；如果基本面恶化，执行止损
- 💡 **记录**：下跌原因、你的反应、后续结果（用于学习）

### 情况2：获利达到目标（例如涨幅30%）
- 👉 **不要贪心**：执行部分卖出计划
- 👉 **建议**：卖出50%，锁定收益
- 👉 **剩余**：让其余50%继续运行，追求更高回报

### 情况3：持仓上涨幅度太大（超过资金的30%）
- 👉 **风险警报**：组合过度依赖单只股票
- 👉 **解决**：卖出部分，用资金配置其他机会
- 👉 **目标**：保持多元化，任何单股不超20%

### 情况4：熊市或大幅下跌
- 👉 **保持理性**：这是测试你投资纪律的时候
- 👉 **检查持仓**：每只都执行止损规则
- 👉 **捡便宜**：用现金对好公司补仓
- 👉 **长期视角**：历史上每次大跌后都有反弹

## 🚨 禁止事项（铁律）

### ❌ 必须禁止
1. **全仓持有** - 一只股票不能超过总资金10%
2. **不设止损** - 每笔投资必须有明确的止损价格
3. **融资炒股** - 用借来的钱投资（风险×10）
4. **期权和衍生品** - 初期完全禁止
5. **盲目跟风** - 没有自己的分析就不买

### 🟡 谨慎使用
1. 集中在单一行业（可以，但不超过50%）
2. 分批加仓（可以，但总投资不超20%止损位）
3. 短线交易（可以尝试，但不超10%资金）

## 🎓 学习案例

### 案例1：典型的失控风险
```
❌ 错误做法：
- 全仓$5,000买入1只股票（TSLA）
- 不设止损，期望"相信价值投资"
- 股票跌30%，亏损$1,500
- 感到恐慌，割肉止损
- 没有记录学到什么

✅ 正确做法：
- 分两次各投$1,000 + 保留$1,000现金
- 设置止损价格，例如$92（基于$100买入价）
- 关注基本面，不被短期波动影响
- 如果触及止损，有纪律地执行
- 记录：为什么止损？这告诉我什么？
```

### 案例2：获利管理
```
初始投资：$1,000买入MSFT
6个月后：上涨到$1,300（+30%）

❌ 贪心做法：
- 继续持有，希望继续涨
- 涨到$1,400后回调到$1,200
- 后悔没有在高点卖出

✅ 理性做法：
- 卖出50%（$650），锁定$300利润
- 剩余$650继续持有，追求更高回报
- 若跌回$950，仅损失$50，总体仍盈利
- 若再涨$100，还能赚$100
- 这样既保证盈利，又捕捉上升空间
```
