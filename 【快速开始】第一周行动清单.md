# 快速开始指南 - 第一周行动清单

> **目标**：帮助你在第一周快速上手系统，建立基本的研究习惯

---

## 📌 今天要做的事（建立系统）

### 第一步：理解系统结构（15分钟）
```
□ 打开和阅读这些文件：
  1. 【系统说明】美股研究框架.md（总体框架）
  2. 本文件（快速开始指南）
  
□ 理解文件夹的作用：
  - 财报分析：学习如何分析公司财务数据
  - 个股研究：完成具体股票的研究分析
  - 交易记录：记录你的交易和复盘
  - 每日晨报：每日市场观察和持仓追踪
```

### 第二步：学习核心知识（1-2小时）
```
□ 按顺序阅读这些文件：
  1. 【学习路线】投资知识体系.md
     - 重点：理解P/E、ROE、负债率等关键指标
     - 时间：1小时
  
  2. 【风险管理】资金配置和止损规则.md
     - 重点：理解为什么止损这么重要
     - 时间：30分钟
  
  3. 【入门指南】纳斯达克基础知识.md
     - 重点：了解纳斯达克的特性和适合投资的公司
     - 时间：30分钟
```

### 第三步：实践基础操作（30分钟）
```
□ 打开Yahoo Finance (finance.yahoo.com)
  - 搜索"QQQ"（纳斯达克100 ETF），看看今天的涨跌
  - 看看AAPL、MSFT、NVDA的当前股价

□ 打开SEC Edgar (sec.gov/edgar)
  - 搜索一个你感兴趣的公司
  - 查看它最新的季度财报(10-Q)

□ 记录下来：
  - 纳斯达克100今日涨幅：_____
  - AAPL当前股价：$_____
  - MSFT当前股价：$_____
  - NVDA当前股价：$_____
```

## 📅 第1周日程

### Day 1 - 今天（建立基础）
```
早上（1小时）：
  □ 阅读系统框架和快速开始指南
  □ 理解投资系统的全貌

下午（1小时）：
  □ 学习财务基础知识
  □ 记录3家公司的基本信息

晚上（30分钟）：
  □ 复述今天学到的内容给自己听
  □ 列出明天的学习计划
```

### Day 2 - 财务知识深入（2小时）
```
目标：掌握如何读财报

内容：
  □ 学习资产负债表、利润表、现金流表的含义
  □ 理解P/E、ROE、负债率等核心指标
  □ 用AAPL作为案例，计算这些指标

任务：
  □ 打开Yahoo Finance查看AAPL的财务数据
  □ 手工计算AAPL的P/E、ROE、负债率
  □ 与行业平均值对比

记录：
  AAPL的主要财务指标：
  - P/E：_____ （行业平均：_____）
  - ROE：_____ （行业平均：_____）
  - 负债率：_____ （行业平均：_____）
  - 评价：AAPL相比行业是便宜/平价/昂贵
```

### Day 3 - 风险管理（1.5小时）
```
目标：理解为什么风险管理这么重要

内容：
  □ 学习止损的意义和执行方法
  □ 理解资金配置的规则
  □ 学习风险管理的案例

任务：
  □ 根据$5,000资金，规划你的资金配置
  □ 选择一个假想股票，模拟制定止损位

记录：
  我的资金配置：
  - 现金储备：$_____ (20%)
  - 核心持仓：$_____ (60%)
  - 学习持仓：$_____ (20%)
  
  假想交易计划：
  - 买入股票：_____
  - 买入价：$_____
  - 止损价：$_____ (亏损__%)
```

### Day 4 - 选择目标公司（2小时）
```
目标：选择3家你想深入研究的公司

步骤：
  □ 列出你熟悉或感兴趣的纳斯达克公司（10个）
  □ 根据这些标准筛选：
    - 收入增长：>10%为好
    - 净利率：>15%为好
    - 负债率：<60%为好
    - P/E：<30倍为好
  □ 选出最符合的3个

建议的3个初学者公司：
  - 第一个：MSFT (稳健的软件巨头)
  - 第二个：AAPL (人尽皆知的消费科技)
  - 第三个：NVDA (高增长的芯片龙头)

你的选择：
  公司1：_____ （为什么选这个？）
  公司2：_____ （为什么选这个？）
  公司3：_____ （为什么选这个？）
```

### Day 5 - 深度分析第一家公司（3小时）
```
目标：完成第一家公司的完整分析

步骤：
  □ 复制【模板】个股分析表.md
  □ 逐项填写该公司的数据（从Yahoo Finance获取）
  □ 计算所有财务指标
  □ 估算内在价值
  □ 写出投资结论

保存：
  文件名：【分析】MSFT(或其他).md
  位置：个股研究文件夹

时间分配：
  - 数据收集：30分钟
  - 财务计算：1小时
  - 竞争分析：1小时
  - 估值和结论：30分钟
```

### Day 6 - 晨报训练（1小时）
```
目标：开始写每日晨报，培养市场感知

步骤：
  □ 复制【框架】晨报模板.md
  □ 今天填写今日的晨报内容
  □ 记录QQQ、AAPL、MSFT、NVDA的涨跌

建议：
  - 选在美股开盘后1-2小时写（这时市场形成共识）
  - 或者晚上美股收盘后写（总结一天）

  □ 市场热点：_____

  □ 你的想法：_____
```

## 🚀 第二周预览

完成第一周后，第二周你将：

1. **完成其他2家公司的分析**
   - 使用同样的模板，但会快很多（可能只需2小时/家）
   - 学会对比不同公司的财务指标

2. **建立观察名单**
   - 选择5-10家感兴趣的公司
   - 记录买入目标价格和理由

3. **制定首次投资计划**
   - 选择第一个要买的公司
   - 设定买入价、数量、止损价

4. **模拟交易**
   - 假设你买了这只股票
   - 每天观察和记录价格变化
   - 体验持仓的心理感受（重要！）

5. **开始写日报**
   - 每周3次或5次写晨报
   - 记录市场动向和你的想法
