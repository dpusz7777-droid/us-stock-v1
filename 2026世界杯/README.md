# 🏆 2026世界杯综合分析系统

## 📋 功能概述

多维度比赛分析工具，综合以下数据源为世界杯投注提供参考：

| 维度 | 权重 | 说明 |
|------|------|------|
| 近期状态 | 20% | 球队近5场战绩评分 |
| 伤病情况 | 15% | 伤病球员影响评估 |
| 赔率变动 | 15% | 博彩市场资金流向 |
| 市场交易量 | 15% | 大额投注信号 |
| 历史交锋 | 10% | 两队过往交手记录 |
| 新闻情绪 | 10% | 媒体和社交媒体情绪 |
| 主场优势 | 10% | 是否为中立场地 |
| 天气影响 | 5% | 比赛日天气条件 |

## 🚀 使用方法

```bash
# 进入目录
cd 2026世界杯

# 运行交互式分析系统
python main.py
```

## 📁 文件结构

```
2026世界杯/
├── main.py              # 主程序入口（交互式菜单）
├── config.py            # 配置（权重、分组等）
├── analyzer.py          # 综合分析引擎
├── team_data.py         # 球队数据管理
├── odds_analyzer.py     # 赔率分析
├── news_sentiment.py    # 新闻情绪分析
├── weather_analysis.py  # 天气影响分析
├── data/                # 数据存储目录
└── analysis_report.md   # 自动生成的分析报告
```

## 📊 手动输入数据示例

```
1. 添加球队数据
   - 录入近期战绩 (W/W/L/D/W)
   - 录入伤病名单

2. 每日更新赔率
   - 从各大博彩平台获取最新赔率
   - 录入系统追踪变动趋势

3. 录入情报
   - 球队新闻发布会信息
   - 球员伤病更新
   - 天气变化

4. 运行分析
   - 系统自动综合评分并生成报告
```
